
# QA Manual Testing Portfólio

## About This Project

This repository contains a complete **Manual QA Testing Portfólio**, developed to demonstrate practical skills in software quality assurance.

The project covers:

* Test design techniques (Equivalence Partitioning and Boundary Value Analysis)
* Structured test scenarios using BDD (Gherkin)
* Business rule validation
* Bug reporting
* Test documentation following real-world standards

---

## Objective

The main goal of this project is to simulate a real QA environment, applying best practices to ensure software quality, reliability, and consistency.

This portfolio demonstrates the ability to:

* Analyze and interpret business requirements
* Develop effective test scenarios.
* Identify edge cases and invalid inputs
* Report bugs clearly and objectively
* Organize test artifacts in a professional structure
